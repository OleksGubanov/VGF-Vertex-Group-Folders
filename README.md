**VGF: Vertex Group Folders**

**VGF** is a tool designed to solve the chaos of large **vertex group** lists by allowing you to organize them into a comfortable **folder** structure. It creates a brand-new panel right inside the standard **Vertex Groups** section. You can use it as a complete replacement for the default list whenever you need to clean up your workspace. If you are tired of scrolling through an endless list of items, **VGF** will help you bring order to your model.

Key Features:

**Folder Organization**
Create folders and quickly distribute your **vertex groups** among them. Every element has a simple dropdown menu on the right side. With one click, you can move a group into a folder, or place a folder inside another folder.

**Order Control**
Move folders and individual groups up or down one step at a time to build your own custom layout. When you move a folder, all items inside it move together automatically. Items move only within their own folder level. You can also close folders to save screen space.

**Native Integration**
The add-on works as a clean wrapper around the default Blender system. All standard tools like Assign, Remove, Select, and Auto Normalize work perfectly from this new interface.

**One-Click Synchronization**
If your 3D model already has **vertex groups**, just click the **Sync From Object** button to load them into the folder view. If you create new groups later using modifiers or other add-ons, click the button again to safely import them without breaking your custom setup.
